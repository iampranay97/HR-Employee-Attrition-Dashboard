# HR Employee Attrition Dashboard

This project is an interactive Power BI dashboard created to analyze employee attrition trends, workforce demographics, job satisfaction, and HR insights.

## Tools Used
- Power BI
- DAX
- Data Visualization

## Dashboard Features
- Employee Attrition Analysis
- Department-wise Insights
- Job Satisfaction Metrics
- Demographic Analysis
- Interactive Slicers & Filters
- KPI Cards & Charts

## Key Insights
- Identified attrition trends across departments
- Analyzed employee demographics and job roles
- Created interactive HR analytics visualizations
